Debian10搭建博客系统（一）

本次学习搭建博客系统

1.安装 git和node.js (root权限操作,其他账户加上 sudo)

本次安装的是12.x版本的，如果需要安装其他版本，只需要把 12.x 换成别的即可。
安装完成后可以输入 node -v 和 npm -v 分别查看node和npm的版本。hexo对node.js有版本要求，根据要求下载对应版本，一般安装最新版本均可以支持。

2.使用npm安装hexo并配置服务器

上面命令中 myblog 可以换成自己喜欢的名字，命令执行后会在当前目录下生成myblog博客主文件夹，hexo server 命令执行后启动博客系统，此时在浏览器输入localhost:4000如果能够访问证明安装成功了。如果在VPS服务器上执行还需要在服务器控制台上开启对应的防火墙端口（4000）。
